# ⌨️ Typing Speed Game

A simple typing speed game built with JavaScript.  
The player types words within a limited time based on the selected level.

## 🚀 Features

- Multiple difficulty levels  
- Timer system  
- Score tracking  
- Save level using localStorage  

## 🛠 Technologies Used

- HTML  
- CSS  
- JavaScript(Vanilla) 

## 📌 Goal

Practice JavaScript logic, DOM manipulation, and working with timers.
